# Fishing Logbook

A simple browser-based fishing logbook created to register catches, fishing conditions, and generate organized PDF reports.

## About The Project

I recently started fishing and decided to build this project as a learning experience and personal test. The idea was to create a lightweight web application capable of registering fishing sessions directly in the browser.

This project is just the beginning. In the future, I plan to transform it into a full Android application and experiment with MIT App Inventor for mobile prototypes.

## Features

- Fish catch registration
- Weight and size tracking
- Bait registration
- Weather conditions
- Fishing location and time
- Catch photo upload
- PDF report generation
- Simple and responsive interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jsPDF

## Future Improvements

- Android version
- GPS integration
- Fishing statistics dashboard

## How to Use

1. Open the `index.html` file
2. Fill the fishing session information
3. Upload a catch photo
4. Click `SAVE RECORD AND GENERATE PDF`
5. The PDF report will be generated automatically
